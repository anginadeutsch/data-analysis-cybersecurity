  Анализ киберугроз 2015–2024

Краткое исследование инцидентов: объем атак по странам, динамика по годам, отраслевой профиль и топ‑10 самых затратных инцидентов. Все визуализации отображаются прямо в ноутбуке (файлы изображений не сохраняются).

   Содержание
- Описание
- Данные
- Установка
- Запуск
- Результаты
- Структура проекта

   Описание
Цель — показать:
- какие страны чаще сталкиваются с атаками,
- как менялась активность по годам,
- какие типы атак характерны для отраслей,
- какие инциденты наиболее затратны.

   Данные
- CSV: `data/Global_Cybersecurity_Threats_2015-2024.csv`
- Поля: `Country`, `Year`, `Attack Type`, `Target Industry`, `Financial Loss (in Million $)`, `Number of Affected Users`, и др.
- Ущерб — в млн $.

   Установка
pip install -r requirements.txt
Запуск
1) Открой ноутбук:
   - локально: `jupyter notebook notebook/cybersecurity-notebook.ipynb`
   - или в VS Code/Colab/Kaggle.
2) Выполни все ячейки (Restart & Run All). Визуализации появятся внутри ноутбука. 

   Результаты
- В ноутбуке есть раздел с краткими выводами по каждому графику и общей картиной.
- Метрики: тренды по годам, распределение по странам и отраслям, топ‑10 по ущербу.
